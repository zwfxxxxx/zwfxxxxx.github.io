100127. 给小朋友们分糖果 II

链接

题目

给你两个正整数 n 和 limit 。

请你将 n 颗糖果分给 3 位小朋友，确保没有任何小朋友得到超过 limit 颗糖果，请你返回满足此条件下的 总方案数 。

示例 1：

输入：n = 5, limit = 2
输出：3
解释：总共有 3 种方法分配 5 颗糖果，且每位小朋友的糖果数不超过 2 ：(1, 2, 2) ，(2, 1, 2) 和 (2, 2, 1) 。
示例 2：

输入：n = 3, limit = 3
输出：10
解释：总共有 10 种方法分配 3 颗糖果，且每位小朋友的糖果数不超过 3 ：(0, 0, 3) ，(0, 1, 2) ，(0, 2, 1) ，(0, 3, 0) ，(1, 0, 2) ，(1, 1, 1) ，(1, 2, 0) ，(2, 0, 1) ，(2, 1, 0) 和 (3, 0, 0) 。

提示：
1 <= n <= 106
1 <= limit <= 106

思路

合法方案数 = 总方案数 - 不合法方案数（至少一位小朋友糖果数 > limit）
总方案数 ： C(n + 2, 2) # 隔板法

三个小朋友分别为甲乙丙
A 甲 > limit方案数 C(n-(limit + 1), 2)
B 乙 > limit方案数 C(n-(limit + 1), 2)
C 丙 > limit方案数 C(n-(limit + 1), 2)
A∩B 甲 > limit 并且 乙 > limit C(n-2 * (limit + 1), 2)
A∩C 甲 > limit 并且 丙 > limit C(n-2 * (limit + 1), 2)
B∩C 乙 > limit 并且 丙 > limit C(n-2 * (limit + 1), 2)
A∩B∩C 甲乙丙都>limit C(n-2 * (limit + 1), 2)(

不合法方案数为 A B C的并集 A∪B∪C = A + B + C - A∩B - A∩C - B∩C + A∩B∩C
因为A B C是相等的 A∩B A∩C B∩C 是相等的
即不合法方案数 = 3 * C(n-(limit + 1) + 2, 2) - 3 * C(n-2 * (limit + 1) + 2, 2) + C(n-2 * (limit + 1) + 2, 2)

合法总方案数 = C(n + 2, 2) - (3 * C(n-(limit + 1) + 2, 2) - 3 * C(n-2 * (limit + 1) + 2, 2) + C(n-2 * (limit + 1) + 2, 2))
